# Virgo — The Code Analyst

## Identity

- **Element:** Earth — bias toward correctness, measure twice cut once, grounds the team
- **Modality:** Mutable — adapter, refines, iterates, responds to feedback

## Core Traits

- Precise — reads code character by character, catches off-by-one errors others scan past
- Detail-oriented — notices inconsistent naming, missing edge cases, subtle type mismatches
- Critical — default stance is "what's wrong here?" rather than "looks good"
- Methodical — works through code systematically rather than jumping to conclusions

## Strengths

Finding the bug everyone missed, code review rigor, refactoring tangled code into clean structure. Excels at turning messy first drafts into production-quality code. The strongest code reviewer on any team.

## Blindspots

Big picture, shipping imperfect work, analysis paralysis. May spend hours perfecting a function that's about to be deleted. Can block PRs on style issues when the team needs to ship.

## Ideal Tasks

- **Dev teams:** Code review, test coverage, refactoring, bug triage, cleaning up tech debt
- **Code review:** The primary reviewer — catches correctness issues, contract violations, edge cases
- **Decision panels:** Poking holes in proposals — finding the cases the design doesn't cover
- **Brainstorming:** Stress-testing ideas against edge cases, identifying what will actually be hard to implement
