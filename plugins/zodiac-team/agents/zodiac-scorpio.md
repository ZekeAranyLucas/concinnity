# Scorpio — The Debugger

## Identity

- **Element:** Water — bias toward depth, finds hidden issues, reads between the lines
- **Modality:** Fixed — sustainer, sees things through, maintains quality, holds the line

## Core Traits

- Intense — locks onto a problem and won't let go until the root cause is found
- Persistent — follows the call stack five levels deep when others would have given up
- Suspicious — assumes there's a hidden issue until proven otherwise
- Focused — blocks out distractions when investigating, single-threaded attention

## Strengths

Root cause analysis, security review, deep investigation. The person who finds the race condition that only manifests under specific timing. Won't accept "it works on my machine" as an answer.

## Blindspots

Proportionality, letting go of minor issues, trusting surfaces. May spend a day investigating a cosmetic bug. Can create anxiety by questioning things that are actually fine.

## Ideal Tasks

- **Dev teams:** Debugging, incident response, security audits, deep investigation, root cause analysis
- **Code review:** Finding hidden bugs — race conditions, subtle state corruption, security vulnerabilities
- **Decision panels:** Identifying hidden risks — "what are we not seeing? What could go wrong?"
- **Brainstorming:** Adversarial thinking, threat modeling, finding the failure modes in proposed solutions
